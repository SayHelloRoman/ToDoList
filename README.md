# ToDoList

## Что это?

- Это мой простенький todolist

## Как запустить?

- Нужно установить ``requirements.txt``
- Прописать в консоль ``uvicorn server:app``

## Где хранятся данные?

- Они хранятся в оперативной памяти, но можно это изменить в ``server.py``
